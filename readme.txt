Easily add a dark mode toggle button to your WordPress site for your visitors. Lightweight and simple!

== Description ==

**Dark Mode Mysite** allows your website visitors to toggle between light and dark modes using a floating button. 
No coding required — just install, activate, and use the shortcode `[dark_mode_button]` anywhere on your site.

**Features:**
- Simple dark/light mode switch
- Saves user preference using localStorage
- Customizable toggle button
- Shortcode support for easy integration
- Works with most themes

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/dark-mode-mysite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Use the shortcode `[dark_mode_button]` in any post or page to show the toggle button.
4. Enjoy dark mode functionality!

== Frequently Asked Questions ==

= Does this plugin affect the admin dashboard? =
No, it only applies to the frontend (public view) of your website.

= Will it work with any theme? =
Yes, it is designed to work with most themes that follow WordPress standards.

= Can I customize the button style? =
Yes. You can edit the CSS file `dark-mode-style.css` to match your site’s design.
